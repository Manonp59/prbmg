API Database

The API Database enables the use of incident data from our database throughout the entire project. It is developed using the FastAPI framework.

Authentication

The REST API designed to provide project data is protected by an API key required for all requests. The verification of this API key is handled by the verify_api_key() function.

@app.middleware("http")
async def verify_api_key(request: Request, call_next):
    """
    A middleware function that verifies the API key in the request headers.

    Parameters:
        request (Request): The incoming HTTP request.
        call_next (callable): The callback function to proceed with the request handling.

    Returns:
        The HTTP response after verifying the API key.
    """
    if request.url.path != "/docs" and request.url.path != "/openapi.json":
        api_key = request.headers.get("X-API-Key")
        if api_key != API_DATABASE_SECRET_KEY:
            raise HTTPException(status_code=401, detail="Accès non autorisé")
    response = await call_next(request)
    return response
